## 技术细节

### 后端搜索实现
```python
if q:
    search_term = f"%{q}%"
    query = query.filter(
        (InspirationItem.category.ilike(search_term)) |
        (InspirationItem.sub_category.ilike(search_term)) |
        (InspirationItem.prompt_template.ilike(search_term))
    )
```

### 前端搜索逻辑
```typescript
useEffect(() => {
  const params = new URLSearchParams();
  if (searchQuery) {
    params.append('q', searchQuery);
  } else if (activeCat) {
    params.append('category', activeCat);
  }
  fetch(`/api/v1/inspirations?${params}`)
    .then(r => r.json())
    .then(data => setItems(data));
}, [activeCat, searchQuery]);
```
